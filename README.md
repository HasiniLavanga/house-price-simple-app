# 🏠 House Price Prediction & Valuation Tool

An interactive Streamlit-based web application that estimates house prices using property features such as overall quality, living area, garage capacity, basement area, and year built.

## 🚀 Features

- Interactive user interface
- Real-time house price estimation
- Property feature analysis
- Streamlit deployment ready

## 🛠️ Technologies Used

- Python
- Streamlit
- Pandas
- NumPy

## ▶️ Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
